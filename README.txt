Instruções para rodar o código
1. Alterar no código o TAM_SOLUCAO definido por um #define no topo do código, colocar o tamanho de acordo com o arquivo que será tratado, exemplo para pcb1173
  #define 1173

2. compilar com
> gcc -o main main.c

3. executar com (nome do arquivo.tsp em seguida)
> ./main.exe pcb1173.tsp
